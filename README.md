# Box Rate Calculator

## Project Overview
The Box Rate Calculator is a web application designed to streamline the corrugated box manufacturing process. It offers precise pricing calculations, customer-specific data management, and the ability to generate job cards quickly. This tool is tailored for manufacturers, enabling them to enhance efficiency and accuracy in their operations.

## Installation
To set up the Box Rate Calculator locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd box-rate-calculator
   ```

2. Open `index.html` in your preferred web browser.

No external installations are required, as this project primarily consists of HTML, CSS, and JS.

## Usage
Once you have opened `index.html`, you will have access to the following functionalities:

- **Cost Calculation**: Input the specifications of your box to get the precise cost based on various factors.
- **Product & Customer Management**: Easily manage product details, assist customers directly, and make batch updates.
- **Job Card Generation**: Create comprehensive job cards for your customers' orders with just a few clicks.

## Features
- **Cost Calculation**: Automatic pricing adjustments based on input parameters.
- **Product & Customer Management**: User-friendly interface for updating product information and managing customer data.
- **Job Card Generation**: Quickly produce job cards with all necessary details, helping to keep operations organized.

## Dependencies
This project uses the following dependencies:
- **Tailwind CSS**: For styling the HTML elements.
- **Font Awesome**: For icons used throughout the application.

These libraries are included via CDN links in the HTML file.

## Project Structure
The project has the following structure:

```
